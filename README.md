# Rain Prediction in Australia
This project builds a machine learning model to predict whether it will rain tomorrow using Australian weather data. The Jupyter notebook covers the complete data science workflow: exploratory data analysis (EDA), handling missing values, data visualization, and feature analysis to prepare for predictive modeling.

Dataset: 145,460 entries with 23 features including temperature, humidity, wind speed, and rainfall history.

Target Variable: RainTomorrow (Binary Classification)
